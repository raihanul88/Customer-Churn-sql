# Customer Churn Analysis using SQL

## Project Overview
This project analyzes customer churn risk using SQL by examining subscription status, payment failures, and support ticket activity.

## Database Tables
- customers
- subscriptions
- payments
- support_tickets

## Key SQL Concepts Used
- SELECT
- JOIN
- GROUP BY
- HAVING
- Subqueries
- Indexing
- EXPLAIN

## Business Questions Solved
1. Which customers have expired or cancelled subscriptions?
2. Which customers had failed payments?
3. Which customers submitted multiple complaints?
4. Which customers are at the highest risk of churn?

## Technologies Used
- MySQL
- phpMyAdmin
- SQL
